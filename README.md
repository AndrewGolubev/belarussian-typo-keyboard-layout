За аснову ўзятая раскладка Бірмана (https://ilyabirman.ru/typography-layout/) для расейскай (рускай) мовы.
Былі замененыя літары: щ → ў, Щ → Ў, и → і, И → І.
Усе астатнія сымбалі засталіся такімі, як у арыгінальнай раскладцы Бірмана для расейскай (рускай) мовы.

Як усталяваць: запусьціце файл setup.exe.
Пасьля для раскладкі беларускай мовы выберыце Тыпаграфічную раскладку.
Калі нешта не атрымліваецца — перазагрузіце кампутар.
